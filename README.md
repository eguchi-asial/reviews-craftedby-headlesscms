## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
## 参考) https://ja.nuxtjs.org/docs/2.x/get-started/commands/
$ npm run build
$ npm run start

# test
$ npm run test

# generate static project
$ npm run generate
```
